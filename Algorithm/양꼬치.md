#### 내 코드 <br>

```Java
class Solution {
    public int solution(int n, int k) {
        int answer = 0;
        int b;  // 서비스로 받은 음료수의 개수
        
        if(n>=10){
            b = n/10; // 10인분당 1개의 서비스 음료
            answer = (12000*n) + (2000*(k-b));
        }else{
            answer = (12000*n) + (2000*k);
        }
        
        return answer;
    }
}
```
